# Databricks DE Associate Study Track

This study track targets the **current** Databricks Certified Data Engineer Associate exam blueprint for exams taken on or before **May 3, 2026**.

The content is organized into one module per exam section. Each section contains:

- Study guide markdown files
- A section quiz with exam-style multiple-choice questions
- A separate answers file with rationales

## Sections

1. [Section 1: Databricks Intelligence Platform](./section-01-databricks-intelligence-platform/)
2. [Section 2: Development and Ingestion](./section-02-development-and-ingestion/)
3. [Section 3: Data Processing and Transformations](./section-03-data-processing-transformations/)
4. [Section 4: Productionizing Data Pipelines](./section-04-productionizing-data-pipelines/)
5. [Section 5: Data Governance and Quality](./section-05-data-governance-quality/)

## Scope Notes

- This track follows the blueprint in `de-associate-cource.txt` for the version live through **May 3, 2026**.
- Newer blueprint topics introduced for exams on or after **May 4, 2026** are intentionally excluded as study objectives for this track.
- Where Databricks documentation uses newer terminology, the study materials call out the terminology drift inline.

## Primary Source Strategy

The modules are anchored to official Databricks material, including:

- The current exam guide
- Databricks documentation for compute, Auto Loader, Unity Catalog, Lakeflow Jobs, Bundles, Spark UI, and related features
